# Processo obrigatório

Use dados do seed, Playwright e as dimensões 390x844, 360x800, 768x1024 e 1440x900. Capture estado normal, vazio, erro e validação; execute o fluxo principal; corrija problemas e repita até três ciclos.

Proteja visualmente apenas fluxos centrais. Leia `references/review-rubric.md` antes de finalizar.
